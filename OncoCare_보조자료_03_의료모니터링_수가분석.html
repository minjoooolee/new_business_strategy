01

모니터링의 세 가지 유형 — 무엇을 모니터링하는가

의료 모니터링은 수집하는 데이터의 성격에 따라 세 가지로 분류됩니다. 각 유형별로 수가 적용 현황과 온코케어 연계 가능성이 다릅니다.

Type A · 생리학적 데이터

웨어러블 RPM
(Remote Physiologic Monitoring)

심전도, 심박수, 호흡수, 산소포화도, 혈압, 체온, 혈당 등 기기가 자동 측정하는 생체신호. 현재 수가 적용의 핵심 대상.

한국: EX871, E6544 수가 적용 중
미국: CPT 99453/99454/99457/99458

Type B · 비생리학적 데이터

PRO 모니터링
(Patient-Reported Outcomes)

환자가 직접 보고하는 증상, 부작용, 삶의 질, 기능 상태. ePRO(전자적 자가보고) 형태로 수집. 암환자 예후관리의 핵심 데이터.

한국: 직접 수가 없음 (간접 경로 존재)
미국: RTM 코드로 확장 논의 중

Type C · 복합 데이터

통합 모니터링
(Integrated Monitoring)

생리학적 + 비생리학적 데이터를 AI가 통합 분석. 웨어러블 데이터 + PRO + 커뮤니티 활동 데이터를 결합한 차세대 모니터링.

한국: 미래 수가 영역
미국: RPM+RTM 복합 청구 가능

미국 RPM 수가 벤치마크 — 한국의 미래 방향

미국 Medicare는 2018년부터 RPM 수가 코드를 단계적으로 도입해왔으며, 현재 환자 1인당 월 최대 $200 이상의 RPM 수익이 가능한 구조입니다. 한국의 수가 정책이 향후 따라갈 가능성이 높은 레퍼런스 모델입니다.

미국 RPM의 핵심 특징 — 한국과의 차이

미국 CMS는 생리학적 데이터뿐 아니라 비생리학적 데이터(PRO 등) 분석 행위에도 수가를 적용합니다. 또한 RTM(Remote Therapeutic Monitoring) 코드를 통해 약물 복용 순응도, 증상 자가보고 등 환자 보고 데이터에 대한 별도 수가도 존재합니다. 한국은 아직 생리학적 데이터(심전도) 중심이지만, 미국 모델을 따라 PRO 기반 모니터링 수가가 도입될 가능성이 높습니다.

04

PRO 모니터링 수가 가능성 — 암환자 자가보고의 수익화 경로

현재 한국에서 PRO에 직접적인 건강보험 수가는 존재하지 않습니다. 그러나 PRO를 수익화할 수 있는 간접 경로는 이미 열려 있으며, 직접 수가 도입을 위한 제도적 토대가 마련되고 있습니다.

수익화 경로 내용 시기 확실성
① 주치의제 통합수가
내 PRO 포함		 4군(전문관리) 환자의 정액관리료(연 26.4만원)에 PRO 수집·분석 활동이 포함. PRO 자체가 아닌 '환자 관리 행위'로 보상받는 구조. 성과평가 인센티브에서 PRO 품질이 반영될 가능성. 2026.07~ 높음
② 제약사 PRO
데이터 라이선싱		 커뮤니티에서 수집된 PRO 데이터를 비식별화하여 제약사에 라이선싱. RWE 연구 목적. 수가가 아닌 B2B 직접 수익. 즉시 가능 높음
③ 심평원 평가지표
반영		 PRO를 병원 인증·급여기준 평가에 활용하는 방안이 심평원에서 논의 중. 미국·영국·캐나다·호주는 이미 PROMs와 지불보상제도를 연결. 2027~2028 중간
④ RTM형 별도 수가
신설		 미국 RTM(Remote Therapeutic Monitoring) 모델을 참고한 환자 자가보고 데이터 모니터링 수가. 약물 순응도, 증상 보고 등. 2028~2030 낮음~중간
⑤ 디지털 치료제

재택 모니터링 수가 — 현황과 전망

재택 모니터링에 직접적인 건강보험 수가는 현재 존재하지 않으나, 업계 전체가 '재택 RPM 수가 신설'을 최우선 과제로 요구하고 있으며, 제도적 환경이 빠르게 변화하고 있습니다.

현재 상황

재택 RPM 수가 부재

EX871은 병원 내(입원+외래) 한정. 업계(휴이노 등)는 '재택 관리에 대한 보험수가가 없는 실정'이라고 공식 언급. 미국·일본형 재택 RPM 수가 도입이 업계 공통 요구사항.

변화의 신호

제도적 토대 마련 중

① 주치의제 4군 '방문·재택진료' 수가 신설
② 통합돌봄법 AI·IoT 건강확인 서비스 포함
③ 의료 마이데이터 서비스 시행(2025.03)
④ 복지·돌봄 AI 혁신계획 상반기 발표 예정

RPM 시장 동향

시장은 이미 '병원→재택' 확장 중

씨어스는 '입원→외래→재택 통합 모니터링'을 공식 전략으로 발표. 메쥬/동아ST도 병원 전반 → 재택 확장 계획. 제약사-디지털 헬스케어 동맹이 본격화되며 시장 경쟁 가속.

온코케어의 기회

커뮤니티 = 재택 모니터링 채널

기존 RPM 사업자는 디바이스 기반이라 '환자가 기기를 착용해야' 모니터링 시작. 온코케어는 커뮤니티 활동 자체가 PRO 수집이 되는 구조. 디바이스 없이도 모니터링 가능.

06

온코케어 PRO 모니터링 전략 — 커뮤니티 활동 = 데이터 수집

온코케어의 가장 강력한 차별점은 '커뮤니티 활동이 곧 PRO 데이터 수집이 되는 설계'입니다. 환자가 의무감으로 설문을 작성하는 것이 아니라, 커뮤니티에서 자연스럽게 활동하면서 PRO 데이터가 축적됩니다.

온코케어 모니터링 BM — 3계층 수익 구조

Layer 1 · 웨어러블 RPM 수가 (생리학적 데이터)

디바이스 연동 → AI 분석 → 병원 수가 청구 → 플랫폼 이용료

스마트링·웨어러블 패치·바이오넷 등과 연동. MOCA AI 엔진이 이상징후 감지·예측. 병원이 EX871 등 수가를 청구하고, AITRICS에 플랫폼 구독료 지급.

웨어러블 디바이스→ MOCA AI 분석→ 이상징후 알림→ 병원 수가 청구→ AITRICS 구독료
Layer 2 · PRO 모니터링 (비생리학적 데이터)

커뮤니티 활동 → PRO 수집 → 데이터 라이선싱 + 수가 간접 연계

온코케어 커뮤니티에서 자발적으로 축적된 PRO 데이터. 비식별화 후 제약사 RWE 연구에 라이선싱. 동시에 주치의제 통합수가·성과 인센티브와 연계하여 간접 수가 수익 확보.

커뮤니티 활동→ AI PRO 구조화→ 표준 척도 매핑→ 제약사 라이선싱 + 수가 간접 연계
Layer 3 · 통합 모니터링 (복합 데이터 — 차세대)

RPM + PRO + 커뮤니티 = 통합 재택 모니터링 플랫폼

웨어러블 데이터 + PRO + 커뮤니티 활동 + PHR이 하나의 환자 ID로 통합. 재택 RPM 수가가 신설되면 이 통합 데이터 플랫폼이 즉시 수익화 가능한 자산이 됩니다. 기존 RPM 사업자에게 없는 유일한 레이어.

웨어러블+ PRO+ 커뮤니티+ PHR→ 통합 재택 모니터링 플랫폼
Bottom Line
온코케어의 모니터링 수가 전략은 '기다리기'가 아니라 '미리 쌓기'입니다. 재택 RPM 수가가 없는 현재에도, 커뮤니티에서 PRO 데이터를 축적하고, 주치의제 통합수가로 간접 수익을 확보하며, 제약사 라이선싱으로 즉시 현금화합니다. 재택 수가가 신설되는 순간, 이미 축적된 데이터와 플랫폼이 곧바로 수익으로 전환됩니다.
