패러다임 전환 — 수가에서 데이터로

보조자료 #1~3은 정부 수가(B2G) 기반의 수익 모델을 다루었습니다. 보조자료 #4부터는 관점이 전환됩니다. 온코케어가 축적하는 데이터 자체가 상품이 되어, 제약사·CRO·보험사에 직접 판매하는 B2B 데이터 비즈니스 모델입니다. 이것이 AITRICS가 지향하는 Medical Data Platform의 핵심입니다.

The Shift
한쪽에서는 플랫폼을 개발하고, 다른 한쪽에서는 데이터를 구축합니다. 커뮤니티가 유저를 모으는 동안, AI 에이전트가 그 활동을 구조화된 의료 데이터로 전환합니다. 이 데이터가 충분히 쌓이면, 정부 수가 없이도 — 데이터 라이선싱만으로 — 독립적 수익이 발생합니다.

PRO 라이선싱 메커니즘 — 커뮤니티 활동이 곧 데이터 생산

2단계 수익 구조 — PHR 연동 전후로 차원이 바뀐다

Phase 1~2 · PHR 연동 전 (~2027)

자가 보고 기반 수익화

환자가 직접 입력한 프로필, 부작용 경험, 삶의 질 보고. 제약사 콘텐츠 후원, 자가보고 기반 임상시험 매칭, PRO 데이터 라이선싱(연간 계약). 데이터 신뢰도는 중간이지만 시계열 밀도와 커뮤니티 규모로 가치 보상.

병원 공식 데이터 기반 수익화

데이터 BM 6가지 수익원 — 단계적 확장 구조

수익원 01 · PHR 불필요

제약사·재단 콘텐츠 후원

암종별 커뮤니티에 제약사가 정보 콘텐츠를 후원. "○○제약 제공 유방암 표준치료 가이드" 형태. 광고가 아닌 신뢰 기반 스폰서십.

수익원 02 · PHR 불필요

임상시험 매칭 (자가보고)

환우 프로필 기반 적격 임상시험 매칭. 자가 보고 기반이므로 스크리닝 탈락률 30~50%. 건당 수수료 모델로 초기 수익.

수익원 03 · PHR 불필요 · 핵심

PRO 데이터 라이선싱

부작용 패턴·삶의 질 변화 데이터를 비식별화하여 제약사에 라이선싱. 시판 후 조사(PMS) 보조 자료. 연간 계약 기반.

수익원 04 · PHR 필수 · 고가치

PRO + 임상 통합 데이터

병원 검사 수치와 환자 경험이 매칭. 제약사가 별도 RWE 연구 없이 확보 가능. 단가 3~6배 상승.

수익원 05 · PHR 필수

RWE 연구 수탁

제약사 요청으로 특정 레지멘의 실제 부작용 프로파일 추출. 병원 수치 + 환자 보고 매칭 → 비식별 리포트. 프로젝트 단위 계약.

수익원 06 · PHR 필수 · 장기

보험사·동반진단 데이터

암환자 의료비 지출 패턴(보험 상품 설계), 바이오마커별 치료 반응 패턴(동반진단). PHR 20만 유저 이상 확보 후 가능.

Medical Data Platform 로드맵 — 데이터 자산 구축 단계

Layer 1 · Now → 2027

환자 접점 + PRO 축적

커뮤니티로 유저를 모으고, AI가 활동을 PRO로 구조화. 자가보고 데이터 축적. 수가 불필요, 즉시 실행. 제약사 콘텐츠 후원 + PRO 라이선싱으로 초기 현금 확보.

Layer 2 · 2027 → 2028

PHR 연동 + 데이터 통합

V.Doc PHR / 마이헬스웨이 연동. 자가보고 + 병원 공식 데이터 결합. RWE 등급 통합 데이터 플랫폼 완성. 수익 규모 3~4배 점프. 글로벌에도 없는 데이터 자산.

Layer 3 · 2028~

데이터 인프라 완성

커뮤니티 + PRO + 모니터링 + PHR 통합. 글로벌 수준 RWE 데이터 플랫폼. 제약사 파트너십·데이터 라이선싱 본격화. 만성질환 확장·해외 진출.

Data Flywheel

데이터 플라이휠 — 선순환 성장 구조

환자가 커뮤니티에서 활동할수록 PRO 데이터가 축적됩니다. 데이터가 쌓일수록 AI 에이전트의 정확도가 올라갑니다. AI가 정확해질수록 환자 경험이 개선되어 더 많은 환자가 유입됩니다. 이 플라이휠이 돌기 시작하면 — 경쟁자가 따라올 수 없는 데이터 해자(Data Moat)가 형성됩니다.

환자 유입↑→ PRO 데이터 축적→ AI 정확도↑→ 개인화 경험↑→ 재방문·충성→ 데이터 라이선싱 수익↑→ 플랫폼 투자↑→ 신규 유입 가속

핵심 결론 — PRO 라이선싱은 Data Platform의 첫 번째 증명

결론 ①

즉시 시작 가능

PRO 라이선싱은 PHR 연동 전에도, 수가 없이도 시작할 수 있는 유일한 데이터 BM. 커뮤니티가 모이는 순간부터 데이터 자산이 축적되기 시작합니다.

결론 ②

시장은 이미 검증됨

PatientsLikeMe(UnitedHealth 인수), Inspire(1,000억 투자), Flatiron($1.9B 로슈 인수). 환자 커뮤니티 데이터 플랫폼의 가치는 글로벌에서 증명 완료.

결론 ③

PHR 연동 시 폭발

자가보고 + 병원 데이터 결합 = 글로벌에도 없는 수준의 RWE 자산. 수익 3~4배 점프. 이것이 AITRICS의 Medical Data Platform 비전의 핵심.

결론 ④

경쟁 해자(Data Moat)

커뮤니티 + AI 구조화 + 병원 네트워크를 동시에 보유한 플레이어는 국내에 없음. 데이터 플라이휠이 가동되면 후발 진입 불가능한 구조적 해자 형성.

Bottom Line — Medical Data Platform으로서의 온코케어
온코케어는 커뮤니티 서비스가 아닙니다. Medical Data Platform입니다. 커뮤니티는 데이터를 모으는 인터페이스이고, AI 에이전트는 데이터를 구조화하는 엔진이며, PRO 라이선싱은 데이터를 수익으로 전환하는 첫 번째 파이프라인입니다. 한쪽에서 플랫폼을 개발하고, 다른 한쪽에서 데이터를 구축하는 — 이 양쪽의 작업이 하나의 비즈니스 모델로 수렴합니다.
