화면 1. 보고서 자동 생성 (시스템)
사용자 조작 없음 — 매월 1일 09:00 자동 트리거
PoC에서는 관리자가 "생성" 버튼 수동 클릭으로 대체. 자동 트리거는 Phase 1 이후 구현.
// 생성 트리거 IF 전월 스크리닝 마감 AND 집계 완료 THEN 보고서_초안 자동 생성 문서번호 = VDOC-{고객사코드}-{YYYY}-{MM}-{SEQ} 상태 = '초안생성' 알림 발송 → 1단계 담당자 (HR)
섹션내용데이터 소스
시행령 제5조 현황제1~3호 이행 상태 + 상세 수치자동 판정 결과
전사 종합지수점수 + 전월 대비집계 결과
위험도 분포양호/관심/주의/위험 비율집계 결과
부서별 비교부서명 · 정신 · 신체 · 종합 · 상태배지부서별 집계
고위험 조치탐지 · 연계 완료 · 미조치 건수알림센터 데이터
교육 이수 현황이수율 · 이수 인원교육 모듈
핵심 3줄 요약이슈 + 권고사항 자동 생성룰베이스 자동
// 핵심 3줄 요약 생성 룰 줄1: "전사 종합지수 {X}점({전월대비}) '{상태}' {유지/전환}" IF 위험부서 존재 THEN + ", {부서명} {점수}점 하락, 집중 관리 필요" 줄2: IF 고위험_미조치 > 0 THEN "고위험 미조치 {N}건 잔여, 즉시 조치 필요" ELSE IF 부서간_편차 > 15p THEN "{상위부서} 대비 {하위부서} 편차 {N}p, 원인 파악 권고" ELSE "전 부서 건강지수 안정 범위 내 유지" 줄3: "시행령 제5조 이행률 {X}%" IF 미완항목 존재 THEN + ", {항목명} 추가 개선 필요"

입력 필드필수설명

미이행 사유 조건부 미이행 선택 시 필수. 항목당 textarea 입력.

화면 3. 2단계 결재 — 검토 (안전보건담당자)
시행령 제5조 체크리스트 — 검토자 확인 필수
✓
제1호: 반기 점검 실시 및 경영자 보고
시스템 자동 판정: ✓ 완료 (참여율 82%) → 검토자 확인 체크 필수
!
제2호: 고위험자 발견 시 필요한 조치
시스템 자동 판정: ! 일부미완 (미확인 1건) → 상태 확인 후 체크. 제2호 미이행 시 검토 의견 필수.
✓
제3호: 안전보건 교육 실시 여부 점검
시스템 자동 판정: ✓ 완료 (이수율 73%) → 검토자 확인 체크 필수
WHEN 검토자가 "승인 진행" 선택 + 서명 THEN VALIDATE 체크리스트 3개 항목 모두 확인 완료 SET 검토_서명일시 = NOW() SET 보고서_상태 = '승인대기' SEND 알림 → 3단계 경영책임자 WHEN 검토자가 "1단계 반려" 선택 THEN REQUIRE 반려_사유 (textarea, 필수) SET 보고서_상태 = '반려(검토)' SEND 알림 → HR: "보완 요청 사항이 있습니다" // HR은 보완 후 1단계부터 재진행
유효성 검증 규칙
⚠

→ "시행령 제5조 이행 항목을 모두 확인해주세요" · 서명 버튼 비활성화
⚠
제2호 상태 = '미이행' AND 검토의견 = 빈칸
→ "미이행 항목에 대한 검토 의견을 입력해주세요"
